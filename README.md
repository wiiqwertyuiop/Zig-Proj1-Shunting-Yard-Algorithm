# The Shunting Yard Algorithm

Zig implementation of "The Shunting Yard Algorithm". First time using Zig!

## Supports

- Multi digit integers
- Brackets and parentheses
- Addition, Subtraction, Multiplication, Division
- Spaces
- Negative numbers
